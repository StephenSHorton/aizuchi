## Default Permission

Default permissions for aizuchi-audio: allow checking and requesting
microphone permission, starting and stopping a capture session, and
listening to the per-capture event channel.

#### This default permission set includes the following:

- `allow-check-permissions`
- `allow-request-permissions`
- `allow-start-capture`
- `allow-stop-capture`
